# ShishaTime iOS

> 📌 **このリポジトリは技術スキルを示すためのポートフォリオプロジェクトです　動作確認をするにはShishaTimeDemoのスキームででビルドしてください**

シーシャ（水タバコ）セッションの管理を支援するiOSアプリケーションです。炭の準備時間の管理や、セッションの記録・整理機能を提供します。

## 📱 主な機能

- **炭タイマー**: 炭の準備時間やシーシャセッションの時間管理
- **ドキュメント管理**: セッションメモや記録の作成・編集・整理
- **テンプレート機能**: 再利用可能なセッション設定テンプレート
- **テーブル管理**: 異なるシーシャセットアップの整理・追跡
- **設定機能**: 時間間隔、テーブル、テンプレートのカスタマイズ

## 🏗️ 技術構成

### アーキテクチャ
- **SwiftUI**: UI開発フレームワーク
- **TCA (The Composable Architecture)**: 状態管理と単方向データフロー
- **AWS Amplify**: 認証・GraphQL APIなどのバックエンドサービス
- **Swift Package Manager**: モジュラー設計による保守性と テスト容易性の向上

### プロジェクト構造
```
Package/Sources/
├── AppFeature/              # アプリのメインエントリーポイント
├── AppTabFeature/           # タブナビゲーション
├── CharcoalTimerFeature/    # タイマー機能
├── DocumentFeature/         # ドキュメント管理機能
├── SettingFeature/          # 設定画面
├── Domain/                  # ビジネスロジック・データモデル
├── Common/                  # 共通UIコンポーネント・ユーティリティ
└── Extension/               # Swift標準ライブラリの拡張
```

## 🛠️ 開発環境

### 必要要件
- **iOS 17.0+**
- **Xcode 15.0+**
- **Swift 5.10+**
- **Node.js** (AWS Amplifyバックエンド用)

### ビルド・実行方法
1. `ShishaTime.xcworkspace`を開く（⚠️ `.xcodeproj`ではありません）
2. Xcodeのスキーム選択で **`ShishaTimeDemo`** を選択
3. ⌘+R でビルド・実行

> 💡 **ShishaTimeDemoスキーム**: AWS Amplifyバックエンドに接続せず、オンメモリのモックデータで動作確認ができるデモスキームです。認証やバックエンド環境の設定なしにアプリの機能を試すことができます。

## 🎯 技術的特徴

### 設計パターン
- **TCAアーキテクチャ**: 状態管理と副作用の制御
- **依存性注入**: TCAの依存性システムによる疎結合設計
- **エラーハンドリング**: 集約的エラー処理とユーザーフレンドリーなアラート表示
- **厳密並行性**: Swift 6の並行性モデルを採用

### バックエンド（AWS Amplify）
- **GraphQL API**: 型安全なデータ操作
- **データモデル**: Category、Document、Table、Timer、Templateエンティティ

### 品質保証
- **モジュラー設計**: 機能別パッケージ分割によるテスト容易性
- **型安全性**: SwiftUIとTCAによる実行時エラーの最小化
